    Support

    1. A guide to website:

    Our website provides an easy and better way to keep track of all the entries of an outsider or any visitor. We directly save data to the Database for safe work. We can generate a report when there is a need. The first page is our home page. The receptionist has to log in before registration, so automatically the "register" button direct to the login page. After login, the receptionist dashboard will open which have two buttons: register and search. Register button will direct to a registration page to register a new visitor while the search button will be used for update and delete. There is also an admin login which redirects to the admin dashboard after login. It contains many buttons including "Generate report", "create new admin", "create new receptionist" and "change password". "Generate report" will generate the report based on selected entities (i.e., Date, Name, Number of visits). "Create new admin" will create the new admin and new receptionist as in "create new receptionist". "Change password" will change the password. Only admin has the right to create new admin/receptionist.

    2. Reset/Change/Forgot password:

    In case of reset/change the password, the admin can directly do it through the admin's dashboard. But in case of forgot password, the user must remember the email id which was given at the time of registration. The email is sent to that id to recover the password.

    3. Delete data:

    The data of a visitor can be deleted easily by the receptionist using the "search" button on the receptionist dashboard. This button will search the visitor whose data is to be deleted, this will direct to the registration page which will be auto-filled and by the use of the "delete" button, the data can be deleted.

    4. Create Admin/Receptionist:

    Only admin has the right to create new admin or receptionist. We have the "create new admin" and "create new receptionist" buttons on the admin's dashboard.
