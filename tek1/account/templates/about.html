    About

    1. Instant Registration using UID:

    Through this web application, it becomes very easy to keep track of all the visitors. It can instantly register the visitor using UID. The receptionist can easily register any visitor and also update the entries of the old visitor.

    2. Hassle-free check-ins:

    The visitor check-ins become very easy and hassle-free. Even the registration form will be auto-filled using the data which was previously scanned by android phones using Aadhar Card. Only a few entries have to be manually written. If the visitor does not have the Aadhar Card (have any other id proof), the whole data can be manually filled.

    3. Generate various kinds of reports:

    Sometimes the record of any visitor may be required for any purpose or the record of particular dates. So the admin in here can generate the reports in the required way using the data stored in the database. The reports can be generated based on three entities "Name", "Date" and "Number of visits".

    4. Prevent Re-registration:

    If a visitor is already registered or he is visiting more than one time to us, then the system automatically prevent the re-registration by fetching the details from database by using phone number and fill the details in registration form.
